Implementation Notes

  • Confidence percentages: Hidden by default, shown on hover via tooltip. This maintains information density while keeping critical probability data accessible.
  • Default selection: "Realistic" (REAL) is pre-selected with blue accent color to indicate it's the base case scenario.
  • Value formatting: Use $XXX.XM format for consistency. Values under $100M show one decimal.
  • Abbreviations: CONS, REAL, OPT, BEST - 4 characters max for alignment.
  • Transitions: 100ms duration for snappy institutional feel.
  • Accessibility: Use role="tablist" and role="tab" with aria-selected for screen readers.
  • Keyboard navigation: Support arrow keys for moving between scenarios, Enter/Space to select.
  • Total width: Approximately 280-320px for standard variant, 220-260px for compact.
